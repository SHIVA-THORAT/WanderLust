## Overview
WanderLust is a full-stack web application inspired by Airbnb, developed to manage property listings and bookings.  
The platform allows users to explore properties, view detailed listings, and make reservations through a secure and user-friendly interface.

---

## Features
- User authentication and authorization
- Create, view, update, and delete property listings
- Browse and search available properties
- Booking and reservation management
- Responsive design for desktop and mobile devices

---

## Technologies Used
- Language: JavaScript
- Frontend: HTML5, CSS3, EJS
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: Session-based / JWT authentication
- Tools & Libraries: REST APIs, Mongoose, Git
